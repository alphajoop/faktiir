**Gestion de factures simple et moderne pour freelances et petites entreprises.**

## Features

- **Factures** : Création, édition, PDF, statuts automatisés
- **Clients** : Gestion complète avec informations détaillées
- **Auth** : Sécurisé avec JWT
- **Design** : Interface moderne, responsive, thème clair/sombre

## Stack

- **Frontend** : Next.js 16 + TypeScript + Tailwind CSS
- **UI** : shadcn/ui + Lucide icons
- **State** : @tanstack/react-query + @tanstack/react-form
- **Backend** : NestJS + Prisma
- **Dev** : Bun + Biome + TypeScript strict

## Contributing
